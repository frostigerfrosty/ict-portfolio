IMS Portfolio - Laurin Gloor

Dieses Projekt wurde im Rahmen des Moduls IDAF IV an der Informatikmittelschule (IMS) Basel entwickelt. Es dient als digitale Visitenkarte zur Präsentation meiner schulischen Laufbahn, technischer Projekte und Qualifikationen.
Technologien

    HTML5

    CSS3 (Grid und Flexbox, Dark/Light-Mode)

    JavaScript

Projektstruktur

    index.html: Startseite

    about.html: Lebenslauf und Interessen

    projects.html: Programmierprojekte

    skills.html: ICT-Kompetenzen

    documents.html: Bewerbungsunterlagen

    /docs/: Ordner mit Dokumenten und dem Bewerbungsdossier als ZIP

    /css/: Stylesheets

    /js/: Skripte
