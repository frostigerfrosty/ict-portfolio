Todo Single Page Application (SPA)

Ein interaktives Webprojekt, entwickelt mit HTML, CSS und nativem JavaScript. Die Applikation strukturiert Aufgaben nach dem Eisenhower-Prinzip (Prioritäten-Matrix) und nutzt das LocalStorage-API zur persistenten Datenspeicherung sowie eine dynamische Client-seitige Suchfunktion.

Code ansehen (GitHub)

Bücherantiquariat

Ein dynamisches Web-Projekt zur Verwaltung eines Bücherantiquariats, entwickelt mit PHP und MySQL. Es beinhaltet eine relationale Datenbankstruktur zur Erfassung des Buchbestands sowie Backend-Logik zur Anzeige, Verwaltung und Filterung der Daten über einen lokalen XAMPP-Server.
